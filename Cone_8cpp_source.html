6 /**
7 * @brief Test ray–cone intersection using the quadratic form of the cone equation.
8 *
9 * Algorithm: Solve for intersection parameter t along the ray.
10 * Ray equation: P(t) = ray.origin + t * ray.direction
11 * Cone surface equation: r_perp = k * h
12 * where h = (P - _apex) · axis_hat (signed distance along axis from apex)
13 * r_perp = |(P - _apex) - h * axis_hat| (distance from axis)
14 * k = tan(half_angle) = _radius / _height (finite)
15 * = _radius (infinite, user-supplied slope)
16 *
17 * Decompose the ray and delta into components parallel/perpendicular to the axis:
18 * D_par = ray.direction · axis_hat
19 * d_par = delta · axis_hat (delta = ray.origin - _apex)
20 * aVec = ray.direction - axis_hat * D_par (perpendicular component of direction)
21 * bVec = delta - axis_hat * d_par (perpendicular component of origin offset)
22 *
23 * Substituting into r_perp² = k² * h² yields a quadratic in t:
24 * A * t² + B * t + C = 0
25 * where:
26 * A = aVec · aVec - k² * D_par²
27 * B = 2 * (aVec · bVec - k² * D_par * d_par)
28 * C = bVec · bVec - k² * d_par²
29 *
30 * Height constraint at candidate t:
31 * h(t) = d_par + t * D_par
32 * - Infinite cone (_height == 0): accept if h > 1e-6 (forward lobe only)
33 * - Finite cone (_height > 0): accept if 0 < h <= _height
34 *
35 * Outward surface normal:
36 * P_perp = hitPoint - _apex - h * axis_hat (vector from axis to hit point)
37 * r_hat = P_perp / |P_perp| (unit radial direction)
38 * normal = normalize(r_hat - k * axis_hat)
39 *
40 * Steps:
41 * 1. Normalize the cone axis
42 * 2. Compute k (slope) from geometry
43 * 3. Compute delta, project onto axis for D_par and d_par
44 * 4. Extract perpendicular components aVec, bVec
45 * 5. Build quadratic coefficients A, B, C
46 * 6. If |A| < 1e-10: ray is parallel to cone surface (degenerate), return no hit
47 * 7. Compute discriminant = B² - 4*A*C; if disc < 0, ray misses
48 * 8. Try nearest root t1 = (-B - sqrt(disc)) / (2*A); fall back to far root t2 if needed
49 * 9. In tryHit: verify t >= 1e-4, compute h, apply height constraint, compute normal, return HitRecord
50 *
51 * @param ray The ray to test against the cone
52 * @return Optional HitRecord if intersection occurs with t >= 1e-4, empty otherwise
53 */
