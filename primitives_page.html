Primitives

Source files: src/Objects/Plugins/CorePlugin/Primitives/

Primitives are renderable shapes. Each one derives from RayTracer::APrimitive and must implement two pure virtual methods:

  • hit(const Ray&) — ray–shape intersection test
  • getBounds() — optional Geometry::AABB for BVH participation

A primitive that returns a finite bounding box from getBounds() is inserted into the RayTracer::BVH; one that returns std::nullopt falls back to the linear traversal path. See Space Partitioning.

Every primitive stores a std::shared_ptr<IMaterial> obtained from its config node via RayTracer::PluginManager.

Sphere

Source: src/Objects/Plugins/CorePlugin/Primitives/Sphere/Sphere.hpp

RayTracer::Sphere is a finite closed surface.

Members:

  • _center — Geometry::Point3D, world-space center
  • _radius — float, sphere radius

Intersection: solves the quadratic |P(t) - center|² = radius² for t.

Bounds: getBounds() returns a finite Geometry::AABB with corners at center ± radius on each axis. **Sphere participates in RayTracer::BVH.**

Plane

Source: src/Objects/Plugins/CorePlugin/Primitives/Plane/Plane.hpp

RayTracer::Plane is an infinite flat surface.

Members:

  • _point — Geometry::Point3D, any point on the plane
  • _normal — Geometry::Vector3D, outward surface normal

Intersection: solves (P(t) - point) · normal = 0 for t.

Bounds: getBounds() returns std::nullopt — the plane has no finite bounding box. **Plane stays on the linear fallback path and never enters the RayTracer::BVH.**

Cylinder

Source: src/Objects/Plugins/CorePlugin/Primitives/Cylinder/Cylinder.hpp

RayTracer::Cylinder is a circular tube with an optional height cap.

Members:

  • _origin — Geometry::Point3D, base center of the cylinder
  • _axis — Geometry::Vector3D, direction along which the cylinder extends
  • _radius — float, tube radius
  • _height — float, total height; a value of 0 means infinite extent

Intersection: solves the ray–cylinder quadratic in the local frame aligned to _axis.

Bounds: getBounds() returns std::nullopt for infinite cylinders (height = 0). Infinite cylinders stay on the linear fallback path; finite cylinders may return a finite Geometry::AABB and participate in the RayTracer::BVH.

Extending with New Primitives

To add a new shape:

  1. Subclass RayTracer::APrimitive.
  2. Implement hit() returning std::optional<HitRecord>.
  3. Implement getBounds() returning a finite Geometry::AABB if the shape is bounded, or std::nullopt if it is infinite.
  4. Register it in a plugin's initializer map. See Plugin System.
