Detailed Description

Built-in plugin providing the camera renderer and core initializers.

The CorePlugin exposes the camera initializer, material factories and primitive constructors used by the default scene loader. Main runtime types: RayTracer::Camera, RayTracer::Lambertian, RayTracer::Phong, RayTracer::Glass.
