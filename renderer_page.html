Renderer

The renderer is the component that turns a RayTracer::Scene into a 2D framebuffer. RayTracer::Camera is the sole concrete implementation of RayTracer::ARenderer in CorePlugin, implementing deterministic path tracing with stratified sampling.

Construction

Two constructor forms are supported:

  • From explicit position and direction vectors — the camera builds its Geometry::TransformMatrix internally.
  • From a pre-built Geometry::TransformMatrix — for callers that have already composed translation, rotation, and scale.

When loaded from a config file the following fields are read via fromNode<T>() (see Scene System):

  • resolution → std::pair<int,int>, framebuffer dimensions
  • position → Geometry::Point3D
  • rotation → Geometry::Vector3D (Euler angles)
  • scale → Geometry::Vector3D (default (1,1,1))
  • fieldOfView → float (default 50)
  • useBVH → bool (default false)

Ray Generation

generateRay(double u, double v) maps normalized screen coordinates (u, v) ∈ [0, 1]² to a world-space Geometry::Ray :

  1. Compute the screen point via Rectangle3D::pointAt(u, v).
  2. Apply _transform to convert from camera space to world space.
  3. Return a Geometry::Ray from the camera origin through the screen point.

Stratified Sampling

Each pixel is divided into a _sqrtSamples × _sqrtSamples grid (default 2 × 2 = 4 samples per pixel). For each cell, the sample point is at the center of that cell in pixel space, producing a uniform stratified pattern with no randomness.

The resulting color samples are averaged to produce the final pixel value.

Path Tracing — castRay

castRay(Ray, Scene, depth) implements the recursive rendering equation:

  1. Call RayTracer::Scene::hit with the current ray. If no intersection, return the background color (black).
  2. Retrieve the Geometry::HitRecord — intersection point, normal, and RayTracer::IMaterial pointer.
  3. Accumulate direct illumination by iterating all lights in the scene and calling RayTracer::ILight::computeLight (shadow rays included).
  4. If depth < _maxDepth, compute the next bounce:
    • direction = material->sample(normal, viewDir) — deterministic outgoing direction
    • weight = material->evaluate(normal, viewDir, direction) / material->pdf(...) — BRDF weight
    • indirect = castRay(bounceRay, scene, depth + 1) — recurse
    • Add weight * indirect to the accumulated color.
  5. Add material->emission() for emissive surfaces.
  6. Return the total accumulated RayTracer::Color.

BVH Integration

When _useBVH = true, RayTracer::Camera::render calls RayTracer::Scene::prepareAccelerationStructure before iterating pixels. Subsequent castRay calls query RayTracer::Scene::hit, which routes bounded primitives through the RayTracer::BVH and unbounded ones through linear search.

See Space Partitioning for the full BVH documentation.
