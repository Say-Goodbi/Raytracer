1/**
2 * @file Matrix.hpp
3 * @brief Fixed-size generic matrix template used across geometry types.
4 *
5 * The Matrix template provides a small, stack-allocated, row-major storage
6 * for floating point values and implements common matrix operations used by
7 * the geometry primitives. It is intentionally lightweight and header-only
8 * so it can be used as a base class for `Point` and `Vector` specializations.
9 *
10 * Template parameters:
11 * - Rows: number of rows (compile-time constant)
12 * - Cols: number of columns (compile-time constant)
13 *
14 * Notes:
15 * - Storage is a plain C-style array of `float` in row-major order.
16 * - Default constructor zero-initializes the contents.
17 * - A copy constructor is defaulted to allow trivial copying.
18 * - `identity()` is only available for square matrices (Rows == Cols).
19 */
